# 🎯 렌즈 왜곡 보정 프로젝트

## 📌 개요
이 프로젝트는 **카메라 캘리브레이션**을 통해 얻은 데이터를 기반으로 **렌즈 왜곡 보정**을 수행하는 프로젝트입니다.  
체스보드 패턴을 이용해 카메라의 내부 파라미터를 추정하고, 이를 이용하여 이미지나 영상에서 발생한 렌즈 왜곡을 보정합니다.

---

## 🔧 사용 기술
- Python
- OpenCV (cv2)
- NumPy

---
